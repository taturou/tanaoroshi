# 棚卸しアプリ (Tanaoroshi App)

スマートフォンのカメラでバーコード（JANコード）をスキャンし、商品の在庫を素早くカウント・記録するためのWebアプリ（PWA）です。

## 🚀 Webサイト（デモ）
https://taturou.github.io/tanaoroshi/

※スマホのブラウザ（Safari/Chrome等）で開き、「ホーム画面に追加」することでネイティブアプリのように全画面で利用可能です。

## ✨ 主な機能
- **高速バーコードスキャン**: Zxingライブラリを使用した1Dバーコード（JANコード）の読み取り。
- **商品情報の自動取得**: Yahoo!ショッピングAPIおよびOpen Food Facts APIと連携し、JANコードから商品名、メーカー名、画像を自動取得。Yahoo!ショッピングAPIで取得できない場合は Open Food Facts からの取得を試みます。
- **手入力フォールバック**: 商品情報を自動取得できない場合でも、そのまま商品名やメーカー名を手入力して登録できます。
- **分類タグ管理**: 商品を「飲料」「食品」などのタグで分類。既存タグの選択や新規タグの作成がその場で可能。
- **既存商品の自動処理**: 過去に登録したJANコードをスキャンすると、自動的に情報を復元し数量を+1します。
- **完全サーバーレス**: データはすべて端末（ブラウザのLocal Storage）に保存されるため、別途サーバーは不要です。
- **CSV入出力**: 記録したデータをCSVで出力したり、外部CSVを読み込んでデータを統合（マージ）したりできます。
- **担当者記録**: 誰が入力したかをCSVに含めることができ、複数人での棚卸し作業に対応。

## 🛠 使用方法
1. **スキャン**: 「スキャン開始」ボタンを押し、商品のバーコードを緑色の枠に合わせます（タップでピント調整が可能）。
2. **登録**: JANコードから取得された情報を確認し、数量を `[+]` `[-]` ボタンで調整して「保存」します。
3. **確認・編集**: 「リスト」タブで登録済みデータを確認できます。検索ボックスで絞り込みや、商品情報の修正・削除が可能です。
4. **出力**: 完了後、「設定」タブからCSVを出力してメールやクラウドストレージで共有します。

## 🔑 事前準備（Yahoo!ショッピングAPIの取得）
本アプリで商品情報の自動取得機能を利用するには、ご自身でYahoo! JAPANのデベロッパーネットワークから Client ID を取得する必要があります。

1. [Yahoo! JAPAN デベロッパーネットワーク](https://developer.yahoo.co.jp/) にアクセスします。
2. アプリケーションID（Client ID）を新規発行します。
3. アプリ内の「設定」タブにある「Yahoo!ショッピングAPI Client ID」欄に、発行されたIDを入力してください。

※IDを設定しなくても、手入力による棚卸し作業は可能です。

## 📦 ビルド・開発方法

### 必要環境
- Node.js (v20以上推奨)
- npm

### インストール
```bash
git clone https://github.com/taturou/tanaoroshi.git
cd tanaoroshi
npm install --legacy-peer-deps
```

### 開発用サーバーの起動
```bash
npm run dev
```

### 本番ビルド
```bash
npm run build
```
`dist` ディレクトリにビルド済みのファイルが出力されます。
